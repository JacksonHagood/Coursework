PPM Image Manipulator
CSCE 121 (adapted partially from assignment)

Program that can perform a library of operations
on .ppm images (no comments).

en.wikipedia.org/wiki/Netpbm#PPM_example
http://paulcuth.me.uk/netpbm-viewer/

Run using ./manipulator.

Example operations:
Rotation
Flipping
Shrinking (frequency or through seam carving)
Tinting
